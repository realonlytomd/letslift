Welcome !

Instructions

Each individualized workout consists of exercises, the weight used, number of sets in each exercise, and the number of repetitions (reps) during each set. The app allows you to keep track of sets completed and the rest time between sets during a workout.

The Process


Log in with your email address and password.
Click: Click here to enter a new workout.
Create a name for your new workout.
Click: Submit
You will see a form to build your first exercise. Enter the required information in the fields.
Click: Submit
To enter more exercises, click: Add More Exercises and repeat the sequence.
When you're finished, click: Finish
Your new workout is then added to the list of available workouts.
Note: You are allowed 5 workouts with 10 exercises each.

Edit

The Edit button shows the full workout of ten exercises with input fields that allow changes to the workout.
Each field is labeled with the current information.
You can add more exercises to your workout in the fields with empty labels (up to a total of ten).
To save your changes for each exercise, click: Submit Changes For This Exercise. The input fields will then display the changes that you made.
Continue editing your exercises, submitting the changes for each exercise.
IMPORTANT: When you complete your editing, you MUST click: Finished Editing at the bottom.
You will return to the list of available workouts.

Delete:

To delete a workout, click: Delete.
IMPORTANT: This action can not be reversed!

Workout

Click on the name of your chosen workout to start your session.
This page shows the Name of your current workout, and then a list of each Exercise Name, the associated number of Sets and Reps, and the assigned Weight.
A row of yellow buttons labelled Reps represent each set for each exercise. Example: Five sets will have five buttons.
Perform your first set. Then click: the first button.
The Reps label of the button will change to the number of assigned reps for that exercise, showing that you have completed the required number of reps.
The button changes color to red to show that you've completed that set.
Note: The clock starts counting your rest time between sets. Pay attention so you don't rest too long! (Android users will hear the elapsed time every thirty seconds.)
When you're ready, complete your next set, and click: the second button. The clock starts over.
Complete your entire workout, checking off each set as you finish them.
Note: If you can't complete the required number of reps, click: the same set button again. The label will count down to however many reps you did complete. If you click the button too many times, just click repeatedly to count down to zero and start over. Stop when it displays the correct number of completed reps.
The clock continues to count while correcting the number of reps completed!

When you are finished with your workout, congratulate yourself!
Click: Workout Finished. You'll be taken back to the list of available Workouts.
Click: Log out.
